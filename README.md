## How to run (PHP Composer and XAMPP required)
1. `git clone git@github.com:Vertipae/PHP-SQL_Quiz_Project.git`
2. `cd PHP-SQL_Quiz_Project`
3. `php composer.phar install`
4. Run with XAMPP
